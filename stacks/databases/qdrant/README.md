# QDrant

Production-ready QDrant Vector Database image.

## Features

- Based on official QDrant image
- Optimized for production use
- Multi-architecture support (amd64, arm64)

## Usage

```bash
docker run -p 6333:6333 zylarian/dockyard-qdrant:latest
```
